# 💰 Cash Flow Tracker

A simple web app to track salary, expenses, and remaining balance with additional features like currency conversion, charts, and PDF export.

---

## 🚀 Features

* Add and manage expenses
* Set monthly salary
* View total expenses and remaining balance
* Budget alert when balance is low
* Currency conversion (INR, USD, EUR, GBP)
* Visual representation using pie chart
* Download report as PDF
* Data saved using localStorage

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js
* jsPDF

---

## 📚 What I Learned

* DOM manipulation and event handling
* Working with `localStorage`
* Managing data using arrays and objects
* API integration using `fetch()`
* Handling asynchronous operations
* Generating PDFs using jsPDF
* Updating charts dynamically with Chart.js
